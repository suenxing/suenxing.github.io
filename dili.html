感谢 周老师 两年的相伴与教诲

  • 哪里的水能丰富
    1. 一般在河流上游
    2. 符合以下条件
      • 降水/冰雪融水丰富，水量大
      • 地势起伏大，河流落差大
  • 哪里的航运价值高
    1. 流经平原，水流平缓
    2. 降水丰富，水量大，水位年际变化小
    3. 无结冰期或结冰期短
    4. 人口稠密，经济发达，运量大
  • 丝绸之路对 沿线国家 的积极影响（注意是沿线国家）
    1. 带动 经济发展
    2. 促进 经济往来
    3. 促进 文化交流
    4. 促进 劳动力就业
    5. 促进 沿线国家之间睦邻友好，国际交往
  • 原料进口多元化的战略意义（从多国进口某原料）
    1. 保障 [工业、粮食等方面]的安全
    2. 缓解 [工业、粮食等方面]的紧张格局
    3. 稳定 [工业、粮食等方面]进口格局，稳定价格成本
  • 河流对沿岸地区的有利影响
    1. 冲击 平原
    2. 提供 水源
    3. 提供 水上航运/水能资源
    4. 提供 水产
  • 东北老工业基地发展时面临的问题
    1. 产业结构单一
    2. 资源枯竭，设备老化，工艺落后
    3. 生态环境恶化
  • 如何解决-东北老工业基地发展时面临的问题
    1. 调节产业结构，发展高新技术产业和第三产业
    2. 部分企业可以更新设备，改进工艺，提高生产效率
    3. 平整地表，植树种草，全面治污，改善生态环境
  • 森林雨林生态效益(如果问“破坏会导致什么”只要反着说就好了)
    1. 调节气候--------------->气候紊乱
    2. 涵养水源，保持水土-->水土流失
    3. 保护生物多样性------->生物多样性锐减
  • 湿地生态效益(如果问“破坏会导致什么”只要反着说就好了)
    1. 防洪抗旱
    2. 涵养水源
    3. 保护生物多样性
    4. 净化水质
  • 湿地生态效益(如果问“破坏会导致什么”只要反着说就好了)
    1. 防洪抗旱
    2. 涵养水源
    3. 保护生物多样性
    4. 净化水质
  • 我国夏季普遍高温(除青藏高原外)，为什么？
    1. 夏季南方地区太阳高度角大
    2. 而北方地区纬度高，白昼时间长
    3. 所以南北地区温度都高
  • 交通线的建设带来的影响
    1. 积极:
      改善交通运输状况，缩短运输时间，促进经济发展
      促进该区域内经济联系和人文交流
      带动旅游等相关产业发展
    2. 消极:
      一定程度上破坏生态环境
      应注重经济效益与自然环境并重
  • 以桥代路的目的
    1. 减少土地资源占用，节约耕地
    2. 保护生态环境
    3. 给野生动物留下迁徙通道
  • 城郊农业发展方向(变化)及分析
    1. 改粮食种植为发展肉蛋奶禽、花卉苗木、农家乐等
    2. 原因:
      靠近城区，消费市场广
      利润高
  • 商品粮基地有利条件
    1. 三江、松嫩：
      地形平坦开阔，集中成片，有利于机械化操作
      地广人稀，余粮多

    2. 鄱阳湖：
      人口稠密，劳动力资源丰富
      消费市场广
  • 人口稠稀的原因（思考方向）
    1. 地形
    2. 气候
    3. 交通
    4. 开发历史
    5. 工农业基础
  • 山区开发，因地制宜
    1. 矿产地区——采矿
    2. 水库河湖——淡水养殖，休闲垂钓
    3. 地势较高——生态林
    4. 地势较低——经济林（林果业）
    5. 山麓平原——种植
    6. 旅游业
  • 欠发达地区如何发展
    1. 矿产丰富，资源优势——采矿等
    2. 风景秀丽，历史悠久——旅游业
    3. 加大开放力度——引进资金、技术、人才
    4. 优化产业结构
  • 生态问题
    1. 东部季风区——水土流失
    2. 西北地区——土地沙漠化
    3. 沿海、过度开采地下水——土壤盐碱化
  • 国际活动的影响
    1. 有利:
      促进交通运输，城市基础设施建设
      促进旅游等服务业发展，帮助劳动力就业
      城市品牌交流推广
    2. 不利:
      场馆难以利用，闲置浪费严重
      维护难度大
